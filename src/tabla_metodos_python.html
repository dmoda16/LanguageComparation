Métodos de Cadenas de Caracteres en Python

Método Descripción Ejemplo de Uso
str.lower() Convierte todos los caracteres de la cadena a minúsculas. 'HELLO'.lower() -> 'hello'
str.upper() Convierte todos los caracteres de la cadena a mayúsculas. 'hello'.upper() -> 'HELLO'
str.capitalize() Convierte el primer carácter de la cadena a mayúscula. 'hello'.capitalize() -> 'Hello'
str.title() Convierte el primer carácter de cada palabra a mayúscula. 'hello world'.title() -> 'Hello World'
str.strip() Elimina los espacios en blanco al inicio y al final de la cadena. ' hello '.strip() -> 'hello'
str.lstrip() Elimina los espacios en blanco al inicio de la cadena. ' hello'.lstrip() -> 'hello'
str.rstrip() Elimina los espacios en blanco al final de la cadena. 'hello '.rstrip() -> 'hello'
str.replace(old, new) Reemplaza todas las apariciones de una subcadena por otra. 'hello world'.replace('world', 'Python') -> 'hello Python'
str.split(sep) Divide la cadena en una lista utilizando un separador. 'hello world'.split(' ') -> ['hello', 'world']
str.join(iterable) Une los elementos de un iterable en una cadena, separados por la cadena actual. ', '.join(['a', 'b', 'c']) -> 'a, b, c'
str.find(sub) Devuelve el índice de la primera aparición de una subcadena, o -1 si no se encuentra. 'hello'.find('e') -> 1
str.index(sub) Devuelve el índice de la primera aparición de una subcadena, lanza ValueError si no se encuentra. 'hello'.index('e') -> 1
str.startswith(prefix) Devuelve True si la cadena comienza con el prefijo especificado. 'hello'.startswith('he') -> True
str.endswith(suffix) Devuelve True si la cadena termina con el sufijo especificado. 'hello'.endswith('lo') -> True
str.isalpha() Devuelve True si todos los caracteres de la cadena son alfabéticos. 'hello'.isalpha() -> True
str.isdigit() Devuelve True si todos los caracteres de la cadena son dígitos. '123'.isdigit() -> True
str.isalnum() Devuelve True si todos los caracteres de la cadena son alfanuméricos. 'hello123'.isalnum() -> True
str.isspace() Devuelve True si la cadena solo contiene espacios en blanco. ' '.isspace() -> True
str.zfill(width) Rellena la cadena con ceros a la izquierda hasta alcanzar el ancho especificado. '42'.zfill(5) -> '00042'
str.format(*args, **kwargs) Formatea la cadena usando sustituciones de campo. 'Hello, {}'.format('world') -> 'Hello, world'
